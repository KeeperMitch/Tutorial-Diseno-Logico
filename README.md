# Tutorial_Diseño_Lógico - Entorno de código abierto para FPGA TangNano 9k
Este repositorio reúne el material necesario para iniciarse en el uso de la FPGA TangNano 9k con herramientas de código abierto. Se abordan desde las configuraciones iniciales hasta ejemplos prácticos de creciente complejidad, además de recursos para estructurar proyectos propios y gestionarlos mediante control de versiones con Git y GitHub.

### Reporte completo del tutorial 
El enlace adjunto contiene un índice detallado que reúne explicaciones, evidencias y la secuencia metodológica aplicada durante el desarrollo del tutorial. Dicho material documenta el uso del entorno de desarrollo de código abierto para la FPGA TangNano 9k, facilitando la comprensión y el seguimiento del proceso.
